The SMS Gateway bridge allows field employees to register themselves by sending an SMS. The smsgateway Android app intercepts incoming SMS on a dedicated SIM card phone and forwards the payload to the LocationWhere backend via POST /api/v1/gateway/sms. The backend's processSmsOnboarding() function creates an employee record and returns a replyMessage for the Android app to send back as confirmation. Duplicate registration (same sender phone) returns status: "duplicate".
