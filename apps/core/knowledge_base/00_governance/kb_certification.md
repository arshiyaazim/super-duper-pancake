# Knowledge Base Certification

Certification confirms that the active Knowledge Base is fit for runtime AI use.

## Certification Checks

- Required metadata exists.
- Runtime-indexed articles are not archived.
- Duplicate content is identified.
- Stale content is listed for owner review.
- Active modules and workflows have KB coverage.
- Sensitive operational secrets are absent.

## Certification Status

Current status is generated by:

```bash
python3 scripts/audit_kb_structure.py
python3 scripts/build_traceability_matrix.py
python3 scripts/generate_certification_reports.py
```
