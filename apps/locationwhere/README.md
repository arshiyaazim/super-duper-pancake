# Employee Monitoring Application (Location Where)

একটি শক্তিশালী এবং সুরক্ষিত কর্মী পর্যবেক্ষণ সমাধান যা ফিনান্সিয়াল কোম্পানিগুলোর জন্য তৈরি করা হয়েছে। এটি কর্মীদের রিয়েল-টাইম লোকেশন, সিম কার্ডের নিরাপত্তা, এবং ডিভাইস ম্যানেজমেন্ট নিশ্চিত করে।

---

## 🏗 প্রজেক্ট স্ট্রাকচার (Project Structure)

প্রজেক্টটি তিনটি প্রধান ভাগে বিভক্ত:

1.  **/backend**: Node.js (Express) এপিআই, PostgreSQL ডাটাবেস এবং Redis ক্যাশ। Prisma ORM ব্যবহার করে ডাটাবেস লেয়ার ম্যানেজ করা হয়েছে।
2.  **/app**: নেটিভ অ্যান্ড্রয়েড অ্যাপ্লিকেশন (Kotlin) যা কর্মীদের ডিভাইসে ইন্সটল থাকে। এটি ব্যাকগ্রাউন্ডে নিরবচ্ছিন্নভাবে কাজ করে।
3.  **/admin-dashboard**: রিঅ্যাক্ট (React) ভিত্তিক আধুনিক ওয়েব প্যানেল যেখান থেকে এডমিন সব কিছু নিয়ন্ত্রণ এবং পর্যবেক্ষণ করেন।

---

## ✨ প্রধান বৈশিষ্ট্যসমূহ (Key Features)

### ✅ এখন পর্যন্ত যা ইমপ্লিমেন্ট করা হয়েছে (Implemented):
*   **অ্যাডাপ্টিভ লোকেশন ট্র্যাকিং:** প্রতি ৬০ সেকেন্ডে রিয়েল-টাইম লোকেশন আপডেট। ব্যাটারি ৩০% এর নিচে নামলে স্বয়ংক্রিয়ভাবে ১২০ সেকেন্ড ইন্টারভালে অ্যাডজাস্টমেন্ট (Power Saving Mode)।
*   **লিগ্যাল কনসেন্ট ও রেজিস্ট্রেশন:** অ্যাপ ব্যবহারের শুরুতে কর্মীদের থেকে ডিজিটাল সম্মতি (Consent) গ্রহণ এবং এমপ্লয়ী কোড ভিত্তিক ডিভাইস রেজিস্ট্রেশন।
*   **জিওফেন্সিং (Geofencing):** নির্ধারিত অফিস বা কাজের এলাকার বাইরে গেলে স্বয়ংক্রিয় অ্যালার্ট জেনারেশন।
*   **সিম পরিবর্তন সনাক্তকরণ:** ডিভাইসে নতুন সিম কার্ড লাগানো হলে বা IMSI পরিবর্তন হলে তাৎক্ষণিক এডমিনকে অ্যালার্ট পাঠানো।
*   **রিমোট কমান্ড:** এডমিন প্যানেল থেকে সরাসরি ডিভাইস লক (Lock), ডেটা মুছে ফেলা (Wipe), সাইরেন বাজানো বা রিমোটলি মেসেজ পাঠানোর সুবিধা।
*   **কল মনিটরিং ও রেকর্ডিং:** ইনকামিং এবং আউটগোয়িং কল লগ স্বয়ংক্রিয়ভাবে সিঙ্ক হওয়া এবং কল রেকর্ড AES-256 এনক্রিপ্টেড অবস্থায় AWS S3-তে আপলোড।
*   **ডিভাইস এডমিন ও অ্যান্টি-ট্যাম্পার:** অ্যাপ আনইনস্টল রোধে ডিভাইস এডমিন পারমিশন এবং রুটেড ডিভাইস সনাক্তকরণের (Root Detection) জন্য RootBeer লাইব্রেরি ব্যবহার।
*   **স্মার্ট সিঙ্ক:** অফলাইন ডেটা লোকাল Room DB-তে জমা থাকে এবং ইন্টারনেট কানেকশন ফিরলে WorkManager-এর মাধ্যমে স্বয়ংক্রিয়ভাবে সিঙ্ক হয়।
*   **অথেন্টিকেশন ও সিকিউরিটি:** JWT ভিত্তিক সুরক্ষিত অথেন্টিকেশন (Access & Refresh Tokens) এবং DataStore ভিত্তিক টোকেন ম্যানেজমেন্ট।

### ⏳ যা এখনও বাকি (Pending / Roadmap):
*   **বিস্তারিত ড্যাশবোর্ড এনালিটিক্স:** এডমিন প্যানেলে কর্মীদের মুভমেন্ট ট্রেইল এবং কাজের সময়ের গ্রাফিক্যাল রিপ্রেজেন্টেশন।
*   **ওয়ার্ক ফোল্ডার মনিটরিং:** কোম্পানির নির্দিষ্ট ফোল্ডারে ফাইল শেয়ারিং এবং এক্সেস ট্র্যাকিং।
*   **অটোমেটেড রিপোর্ট জেনারেটর:** সাপ্তাহিক এবং মাসিক পারফরম্যান্স PDF/Excel রিপোর্ট।
*   **ভয়েস কমান্ড কমান্ড প্রোম্পট:** রিমোটলি ভয়েস কমান্ডের মাধ্যমে নির্দিষ্ট সিকিউরিটি টাস্ক সম্পন্ন করা।

## 🛠 টেকনোলজি স্ট্যাক (Tech Stack)

*   **Backend:** Node.js v20, Express, Prisma ORM, PostgreSQL 16, Redis 7, JWT (Access/Refresh Tokens), Zod.
*   **Android App:** Kotlin, Retrofit, Room DB, Hilt (Dependency Injection), WorkManager, DataStore, Fused Location API.
*   **Dashboard:** React 18, Ant Design 5, TanStack Query, Leaflet Maps (Live Tracking), Zustand.
*   **Cloud & Infra:** AWS S3 (Storage), Firebase Admin (FCM Push), PM2, Nginx.

## 🚀 ইন্সটলেশন গাইড (Installation Guide)

### ১. ব্যাকএন্ড (Backend)
```bash
cd backend
npm install
cp .env.example .env  # আপনার DATABASE_URL এবং Firebase credentials সেট করুন
npx prisma migrate dev
npm run dev
```

### ২. এডমিন ড্যাশবোর্ড (Web Dashboard)
```bash
cd admin-dashboard
npm install
npm run dev
```

### ৩. অ্যান্ড্রয়েড অ্যাপ (Android App)
*   অ্যান্ড্রয়েড স্টুডিওতে `Location_Where` প্রজেক্টটি ওপেন করুন।
*   `app/local.properties` ফাইলে `API_BASE_URL=http://5.189.131.48:3000/` সেট করুন (Retrofit `BuildConfig.API_BASE_URL` থেকে এটি পড়ে)।
*   অ্যাপটি বিল্ড করে ফিজিক্যাল ডিভাইসে রান করুন এবং প্রয়োজনীয় পারমিশনগুলো (Location, Record Audio, Device Admin) নিশ্চিত করুন।

## 🔒 নিরাপত্তা ও গোপনীয়তা (Security & Privacy)
*   **ডেটা সিকিউরিটি:** সব এপিআই কমিউনিকেশন TLS 1.3 এর মাধ্যমে সুরক্ষিত।
*   **এনক্রিপশন:** সেনসিটিভ ডেটা এবং কল রেকর্ডিং AES-256-CBC ব্যবহার করে এনক্রিপ্ট করা হয়।
*   **অথরাইজেশন:** রোল বেজড এক্সেস কন্ট্রোল (RBAC) - Super Admin, HR, এবং Manager দের জন্য আলাদা পারমিশন।
*   **প্রাইভেসি:** কর্মীদের থেকে অ্যাপের শুরুতেই স্বচ্ছভাবে কনসেন্ট গ্রহণ করা হয়।
